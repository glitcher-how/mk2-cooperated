# sbd-drones-economics

Учебный репозиторий для системы управления экономикой полётов БАС. Структура приведена к требованиям по размещению файлов, документации, тестам, Docker-окружению и оформлению Python-кода.

## Что внутри

- `docs/` — общая документация проекта.
- `components/sdk-core/` — переиспользуемые базовые классы и интерфейсы для message-driven сервисов.
- `systems/operator/` — система оператора БАС с gateway и бизнес-компонентом.
- `broker/` — локальная реализация шины сообщений для разработки и тестов.
- `tests/` — интеграционные и e2e тесты уровня репозитория.
- `demos/` — демонстрационные Jupyter notebook.

## Быстрый старт

```bash
make setup
make tests-all
make run-all
```

## Docker

В репозитории добавлены `dockerfile` для сервисов и `docker-compose.yaml` для запуска system-образа.
Профиль `split` оставлен как заготовка для развёртывания отдельных контейнеров после подключения реального брокера сообщений.
